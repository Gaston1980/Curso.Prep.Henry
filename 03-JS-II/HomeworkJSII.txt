* `for`: es un Loops statement , que sirve para ejecutar una funcion o
sentencia determinadas cantidades de veces mientras se cumpla una condicion.
for( declaramos una variable; una expresion condicional; incrementamos la variable) {
    aca escribo mi codigo
}
for( var a = 0; a<100; a++) {
    console.log(a)
}

* `&&`: es un operador logico,  se define como AND/Y 
`||`: es un operador logico, se define como OR/O
 `!`: es un operador logico, se define como NOT/NO: !==; !false; (!(1===1))
